The Project

That fantasy novel you've always wanted to write? That's a project. FictionFolder allows you to organize your projects into separate, organized folders.

   The Text Editor

Write and organize drafts in a basic, distraction-free environment.

   The Plotbuilder

Open, continue, and close plot threads, track plot events, and view project characters.

   The Worldbuilder

Create interactive note documents and search Wikipedia to research narrative settings.

   The Character Designer

Search names, use description and personality templates, and explore advanced character development entry fields.
